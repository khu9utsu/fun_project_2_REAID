## Run Locally

Clone the project

```bash
  git clone https://github.com/khu9utsu/fun_project_2_REAID
```

Go to the project directory

```bash
  cd fun_project_2_REAID
```

Install dependencies

```bash
  pip install streamlit
```

Start the server

```bash
  streamlit run app.py
```
